Comment ça marche

Trois étapes pour des factures conformes

Transformez vos PDF en factures Factur-X en quelques secondes. Notre API fait le travail pour vous.

1

Uploadez votre PDF

Envoyez votre facture PDF existante via notre interface ou notre API REST. Formats supportés : PDF, PDF/A.

2

Ajoutez les métadonnées

Renseignez les informations obligatoires : numéro de facture, SIRET, montants HT/TTC, et dates.

3

Récupérez votre Factur-X

Téléchargez votre PDF enrichi avec les métadonnées XML intégrées. Prêt pour l'envoi.

api-request.sh 
curl -X POST https://api.metron.fr/api/invoices/facturx \
  -H "X-API-Key: votre_cle_api" \
  -F "pdf=@facture.pdf" \
  -F "invoiceNumber=FAC-2026-001" \
  -F "sellerName=Ma Société" \
  -F "sellerSiret=12345678901234" \
  -F "totalHT=1000.00" \
  -F "totalTTC=1200.00" \
  -o facture-facturx.pdf
// Response: 200 OK - PDF Factur-X
